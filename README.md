🎓 E-Learning Platform

A modern, colorful, and fully responsive E-Learning Platform designed for seamless learning across mobile, tablet, and desktop devices. Built with a focus on user experience, performance, and scalability.

🚀 Features
📚 Course browsing and filtering
🎥 Video-based learning system
🧠 Interactive quizzes (MCQs with instant results)
📊 User dashboard with progress tracking
🏆 Achievements, badges, and streak system
🔍 Smart search functionality
❤️ Wishlist / bookmark courses
🔔 Notification system
🌙 Dark mode & Light mode toggle
📱 Fully responsive (Mobile + PC + Tablet)
🎨 UI/UX Highlights
Modern colorful UI with gradients and glassmorphism
Smooth animations and transitions
Clean and intuitive layout
Mobile-first responsive design
Accessible and user-friendly components
🛠️ Tech Stack
Frontend: React
Styling: Tailwind CSS
Animations: Framer Motion
Icons: Lucide
